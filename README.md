# Velvet Aura Beauty Studio

Готовый Flask-проект салона красоты с:
- красивой главной страницей
- портфолио мастеров
- услугами и ценами
- записью по мастерам
- админ-панелью
- панелью мастера
- базой SQLite

## Запуск локально

```bash
pip install -r requirements.txt
python app.py
```

Открой:
`http://127.0.0.1:5000`
